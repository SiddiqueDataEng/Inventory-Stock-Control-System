# Inventory & Stock Control System

**Real-time Inventory Tracking**

## Overview
Real-time inventory tracking system with automated reorder alerts and multi-warehouse stock management.

## Technology Stack (2011)
- **Database**: Microsoft SQL Server 2008
- **Features**: Triggers, Indexing, Real-time Updates
- **Backend**: T-SQL Stored Procedures
- **UI**: Windows Forms Application

## Features
- Multi-warehouse inventory tracking
- Real-time stock updates
- Automated reorder alerts
- Stock movement history
- Low stock notifications
- Inventory aging reports
